The themes below were surfaced by topic modeling every sentence in the references. Hover a theme for its model keywords and source quotes, then click a quote to jump to it in the testimonial below.

how this was built

The green themes aren't hand-picked. A small text-analysis pipeline surfaced them from what people actually wrote:

  1. 1Start with the six recommendations, as plain text.
  2. 2Split each one into individual sentences — about 32 in all.
  3. 3Turn each sentence into a TF-IDF vector with scikit-learn — a standard way to turn text into numbers that up-weights the words that make a sentence distinctive.
  4. 4Factor those vectors with NMF (non-negative matrix factorization), a topic-modeling method, so sentences about the same idea cluster into a theme.
  5. 5Name each theme from the words the model weighted most; its quotes are the sentences that landed in it. These become the green tags above.

Six testimonials is a small corpus, so the model surfaces the themes and I curate the labels — the goal is to show the reasoning, not to over-automate it.
