# Welcome to (the source code) of my personal website
 
This repository contains the source code for my personal website and blog, built with [Quarto](https://quarto.org/).

The site includes blog posts, talks, and project descriptions related to data science, software engineering, and public-interest technology.

## Structure

This repository includes two distinct kinds of material:

- **Website code**  
  Quarto configuration, templates, stylesheets, JavaScript, and build tooling.

- **Content**  
  Blog posts, images, figures, and other authored materials.

These are intentionally licensed **separately** (see below).
